# Students App

This app manages student-related data and operations.

## Features
- **Student Profile**: Stores first name, last name, email, and date of birth.
- **API Endpoints**: Full CRUD support for student records via `/api/students/`.

## Components
- `models.py`: `Student` model definition.
- `serializers.py`: `StudentSerializer` for API data conversion.
- `views.py`: `StudentViewSet` for handling requests.
