# Teachers App

This app manages teacher-related data.

## Features
- **Teacher Profile**: Stores first name, last name, and email.
- **Course Connection**: Serves as the primary entity for course assignments.
- **API Endpoints**: CRUD support via `/api/teachers/`.

## Components
- `models.py`: `Teacher` model definition.
- `serializers.py`: `TeacherSerializer`.
- `views.py`: `TeacherViewSet`.
